# Infrastructure

Implementace doménových interfaces a podpůrná infrastruktura. Balíčky: `config/` (env proměnné), `database/` (SQLite manager, migrace), `sqlite/` (repozitáře), `security/` (JWT, hesla, permissions), `di/` (Wire compile-time DI).
