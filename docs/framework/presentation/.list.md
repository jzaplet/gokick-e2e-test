# Presentation

I/O vrstva aplikace. Veškerá komunikace s vnějším světem prochází přes `presentation/` -- HTTP requesty i CLI příkazy.

Dva vstupní body:

- **`presentation/http/`** -- server, handlery, middleware, response helpery.
- **`presentation/console/`** -- Cobra CLI příkazy (`serve`, `seed`, `create-user`).
